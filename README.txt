# Memory Matching Game

## Overview

A SwiftUI-based memory matching game where players flip cards to find matching pairs. The game features cute animal emojis as cards.

## Features

- Flip cards to reveal animal emojis.
- Match pairs to win the game.

## Technologies Used

- Swift
- SwiftUI

## How to Run

1. Open the project `MemoryGame.xcodeproj` in Xcode.

2. Build and run the project in Xcode to play the game on a simulator or device.
